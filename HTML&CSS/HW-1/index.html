Base

  • git config --global user.name blabla - вказуємо наше ім'я
  • git config --global user.email blabal@bla - вказуємо електронну адресу
  • git init - створює репозиторій
  • git status - показує поточний стан репозиторію
  • git add - додає вміст файлу для подальших комітів
  • git commit - створює коміт
  • git log - показує історію комітів
  • git checkout - повертає до вказаного коміту або гілки
  • git branch - показує всі наявні гілки
  • git push - відправляє локальні зміни до глобального репозиторія
  • git pull - отримання змін з віддаленого репозиторію у локальний
  • git merge - з'єднання гілок в одну
  • git clone - клонування існуючого репозиторію

.gitignore - приховує файли і папки від системи контролю версій Git

Advensed

  • git revert - відміна останього коміта
  • git reset - повертає файл до стану Modified
  • git rebase - переносить зміни з однієї гілки в іншу
  • git cherry-pick - переносить окремі коміти з одного місця в інше
  • git stash - ховає зміни, з можливість використати їх пізніше
